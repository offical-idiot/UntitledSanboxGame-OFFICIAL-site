How to Play

  1. Download the game or launch the web version
  2. Open the game
  3. Spawn objects and experiment with physics
  4. Build, destroy, and explore freely

Controls

  • WASD - Move
  • Mouse - Look around
  • Left Click - Interact / Spawn
  • Right Click - Alternate action
  • ESC - Pause Menu

About the Game

Untitled Sandbox Game is an open-ended sandbox where you can experiment with physics, build structures, and create your own chaos. There are no rules—just tools and your imagination.
